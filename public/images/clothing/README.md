# Clothing Images Folder

## Structure

Add your clothing images to these folders:

```
clothing/
├── base/          # Base layer items (t-shirts, thermal tops, etc.)
├── mid/           # Mid layer items (jumpers, cardigans, fleeces)
├── outer/         # Outer layer items (coats, jackets)
└── accessories/   # Accessories (hats, gloves, scarves, socks)
```

## Image Requirements

- **Format**: PNG (with transparency) or JPG
- **Size**: 400x400px to 800x800px recommended
- **Background**: Transparent or white recommended
- **Style**: Simple, clean product photos or illustrations
- **Naming**: Use exact filenames from CLOTHING_ITEMS_LIST.md

## How It Works

- If an image exists, it will be displayed
- If an image is missing, an emoji placeholder will show instead
- The app automatically maps recommendation text to image filenames
- See CLOTHING_ITEMS_LIST.md in the root folder for the complete list

## Tips

- Keep images simple and recognizable
- Maintain consistent style across all images
- Use neutral colors that won't clash with the UI
- Consider using illustrations for a cohesive look

## Image Sources

Free resources for images:
- **Photos**: Unsplash, Pexels
- **Illustrations**: unDraw, Storyset
- **Icons**: Flaticon, The Noun Project
- **AI Generated**: DALL-E, Midjourney, Stable Diffusion
